# Web参与仪表板

通过Web参与控制板，可以了解Web访问者如何与关键内容进行交互。 它可以跨客户行业和地区划分数据，以帮助您了解参与趋势。 使用此信息板可以通过显示Web行为模式来支持战略决策，这些行为模式为内容战略和客户目标定位提供了信息。

要访问&#x200B;_Web Engagement仪表板_，请在左侧导航中选择&#x200B;**[!UICONTROL 仪表板]**&#x200B;项。 然后选择页面顶部的&#x200B;**[!UICONTROL Web参与]**&#x200B;选项卡。

![Web参与仪表板](./assets/web-engagement-dashboard.png){width="700" zoomable="yes"}

## 筛选数据

单击左上角的&#x200B;_筛选器_ （ ![筛选器图标](../assets/do-not-localize/icon-filter.svg) ）图标，可使用以下任意属性筛选显示的数据：

* **[!UICONTROL 帐户区域]** — 按与帐户关联的一个或多个选定地理区域过滤数据。
* **[!UICONTROL 帐户行业]** — 按与帐户关联的一个或多个选定的行业分类筛选数据。
* **[!UICONTROL 日期范围]** — 按选定的日期范围筛选数据。 默认范围为当前日期。

![按属性筛选显示的数据](./assets/web-engagement-dashboard-filters-dialog.png){width="500"}

为要用于筛选数据的每个属性选择多个值，然后单击&#x200B;**[!UICONTROL 应用]**。

## [!UICONTROL 热门页面浏览量] {#top-page-views}

>[!CONTEXTUALHELP]
>id="ajo-b2b_web_engagement_top_page_views"
>title="最多页面浏览量"
>abstract="您网站上浏览次数最多的页面，有助于识别哪些内容带来了最多流量。"

此表显示前10个最常查看的网页，帮助您确定哪些内容对访问者最敏感。 数据包括：

| 列 | 描述 |
| ------ | ----------- |
| 页面名称 | 网页的名称或标题。 |
| 查看总数 | 页面查看的总次数。 |
| 已知访客(%) | 分配给已知（已标识）访客的页面浏览的百分比。 |
| 未知访问者(%) | 归因未知（匿名）访客的页面查看次数的百分比。 |

![热门页面查看次数表](./assets/web-engagement-dashboard-top-page-views.png){width="650" zoomable="yes"}

## [!UICONTROL 按帐户区域划分的页面浏览量] {#page-views-by-region}

* 帐户区域的名称
* 页面查看次数

## 使用数据

若要使用数据，请使用&#x200B;_更多_ (**...**) 每个图表右上角的菜单，然后选择&#x200B;**[!UICONTROL 查看更多]**&#x200B;以查看扩展数据和见解。

显示的弹出窗口包括一个图表和一个显示数据细分的表格。

要下载数据，请单击数据表右上角的&#x200B;**[!UICONTROL 下载CSV]**。

![热门页面查看表查看更多弹出窗口 — 单击下载CSV](./assets/web-engagement-dashboard-top-page-views-view-more.png){width="700" zoomable="yes"}
