# 角色分析仪表板

角色分析功能板提供了有关购买团体角色如何随时间演变和参与的信息。 它有助于营销人员了解角色获取趋势、参与模式以及最近的营销活动如何推动购买组中不同角色的参与。

若要访问此仪表板，请展开左侧导航栏中的&#x200B;**[!UICONTROL 帐户]**，然后选择&#x200B;**[!UICONTROL 购买群组]**。 选择&#x200B;**[!UICONTROL 角色分析]**&#x200B;选项卡。

![角色分析仪表板](assets/roles-insights-dashboard.png){width="800" zoomable="yes"}

仪表板包括三个视图：

| 视图 | 描述 |
| ---- | ----------- |
| [!UICONTROL 随时间变化的角色获取] | 可视化一段时间内添加到购买群组、按角色分段并按月分组的成员数。 |
| [!UICONTROL 按角色参与度] | 显示按购买组角色划分的参与活动总数（例如电子邮件打开次数、点击次数和表单提交次数）。 |
| [!UICONTROL 随时间变化的角色参与度] | 跟踪一段时间内购买团体参与的活动，按角色显示每月参与趋势。 |

## 筛选数据

单击左上角的&#x200B;_筛选器_ （![筛选器图标](../assets/do-not-localize/icon-filter.svg)）图标，以使用以下任一属性筛选显示的数据：

* **[!UICONTROL 角色]** — 按一个或多个选定的购买团体角色筛选数据。
* **[!UICONTROL 解决方案兴趣]** — 按一个或多个选定的解决方案兴趣筛选数据，以重点关注特定的产品线或产品/服务。
* **[!UICONTROL 日期范围]** — 将数据限制在特定时间段内（默认为上一年）。

角色分析的![筛选器对话框](./assets/roles-insights-filter-dialog.png){width="400"}

对于每个属性，选择要用于筛选数据的值，然后单击&#x200B;**[!UICONTROL 应用]**。

## [!UICONTROL 随时间变化的角色获取]

随着营销和销售团队招募新人员、提高参与度并丰富数据，购买组的新成员将获得资格。 此报表显示了这些工作如何转化为按角色细分的新资格购买组成员。 每次将某人添加到购买组时，报告都会在相应的角色下记录该添加。

![角色分析仪表板 — 随时间变化的角色获取](assets/roles-insights-role-aquisition-over-time.png){width="600" zoomable="yes"}

图表上的每条线表示一个角色。 将鼠标悬停在线上的绘图点上可查看详细信息，包括：

* 获取日期
* 收购次数

要查看更多详细信息，请单击右上角的&#x200B;**...**&#x200B;菜单图标。

## [!UICONTROL 按角色参与度]

此报表汇总了选定时间期内按购买组角色列出的参与活动总量。 使用此报告可以查看每个角色如何参与对您的营销工作的响应。

![角色分析仪表板 — 按角色列出的参与度图表](assets/roles-insights-engagement-by-role.png){width="500" zoomable="yes"}

图形上的每个条形图都表示一个角色。 将鼠标悬停在栏上可查看有关所显示计数的详细信息，包括：

* 角色名称
* 参与计数

要查看更多详细信息，请单击右上角的&#x200B;**...**&#x200B;菜单图标。

## [!UICONTROL 随时间变化的角色参与度]

此报表可跟踪一段时间内的购买团体活动，帮助您了解最近的促销活动如何促进不同角色之间的参与。 使用此信息可优化营销策略并更有效地定位特定角色。

![角色分析仪表板 — 随时间变化的角色参与度](assets/roles-insights-engagement-by-role-over-time.png){width="500" zoomable="yes"}

图表上的每条线表示一个角色。 将鼠标悬停在线上的绘图点上可查看详细信息，包括：

* 日期
* 参与计数

要查看更多详细信息，请单击右上角的&#x200B;**...**&#x200B;菜单图标。

## 使用数据

要处理数据，请使用每个图表右上角的&#x200B;_更多_ (**...**)菜单。

### [!UICONTROL 穿透钻取]

对于&#x200B;_[!UICONTROL 按角色]_&#x200B;进行的参与，请选择&#x200B;**[!UICONTROL 穿透钻取]**，以便按角色和解决方案兴趣对参与进行深入分析。

应用于仪表板的全局过滤器会转移。 单击左上角的&#x200B;_筛选器_ （![筛选器图标](../assets/do-not-localize/icon-filter.svg)）图标以[更改穿透钻取视图的属性筛选器](#filter-the-data)。

### [!UICONTROL 查看更多]

选择&#x200B;**[!UICONTROL 查看更多]**&#x200B;以查看扩展数据和见解。

显示的弹出窗口包括一个图表和一个显示数据细分的表格。
