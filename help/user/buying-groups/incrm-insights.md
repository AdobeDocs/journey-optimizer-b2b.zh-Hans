# In-CRM Insights

Marketo In-CRM Insights是一个集成到Salesforce中的基于Web的应用程序，通过它，您可以直接在Salesforce中访问AJO B2B购买小组。 这使您能够发现提高参与度和销售潜力的机会。

In-CRM分析在[Marketo销售分析包](https://experienceleague.adobe.com/zh-hans/docs/marketo/using/product-docs/marketo-sales-insight/msi-for-salesforce/installation/install-marketo-sales-insight-package-in-salesforce-appexchange)中可用。

## 使用In-CRM Insights

In-CRM Insights可通过应用程序启动程序在Salesforce中使用。

1. 单击Salesforce中的应用程序启动器。
1. 选择或搜索`Journey Optimizer B2B Edition`。
1. 在打开的选项卡中，使用您的Adobe凭据登录。
1. 从下拉列表中选择一个托管您的帐户历程的沙盒。

您的购买组已加载并可用。 数据通过In-CRM Insights为只读。

>[!NOTE]
>用户必须是[B2B销售用户](https://experienceleague.adobe.com/zh-hans/docs/journey-optimizer-b2b/user/admin/user-management#add-users-to-a-role)角色的成员才能访问In-CRM Insights。
>

通过选择一个购买组，您现在可以浏览[组详细信息](https://experienceleague.adobe.com/zh-hans/docs/journey-optimizer-b2b/user/accounts/sales-experience/buying-group-details#)，就像在Journey Optimizer B2B中一样。
