# 购买群组

对于B2B销售和营销活动，客户对任何策略都至关重要。 每个帐户都有一组与之关联的人员，这些人员可能是帐户的员工或与帐户一起工作的承包商。 帐户是分层的，不同的产品可能在分层结构的不同级别销售。 例如，Adobe Experience Platform可以在公司级别出售给顶级客户，而Adobe Photoshop可能会出售给代表组织内某个部门或部门的客户，例如大型公司内的设计部门。

![帐户角色关系图](assets/account-roles-diagram.png){width="800"}

在该帐户中，可能有一部分人组成&#x200B;_购买组_。 这些人员最终会做出购买决定，因此他们需要营销人员给予特别关注，并且可能需要将不同的信息提供给他们，而不是与帐户关联的其他人员。 购买组可能由不同产品线或产品的不同人员组成。 例如，网络安全产品通常需要首席信息官或首席安全官以及法律部门的代表来批准购买，但错误跟踪产品通常需要工程部副总和ITDirector作为购买组的成员。

## 关键组件

您可以在Journey Optimizer B2B版本中建立购买组，根据销售团队负责销售的解决方案确定目标客户列表中缺少的成员，从而提高营销效率。 在您和营销团队开始创建购买群组之前，请确保您定义了关键组件。 这些组件对于实现您的业务目标和目的至关重要。

| 组件 | 用途 |
| --------- | ------- |
| 解决方案兴趣 | 此组件提供以下问题的答案： <ul><li>作为营销组织，您销售什么？</li><li>您打算销售什么产品或产品系列？</li></ul>  **_示例：_**&#x200B;向现有客户交叉销售新产品X |
| 帐户受众 | 此组件提供以下问题的答案： <ul><li>你卖给谁？</li><li>您定位的帐户列表是什么？</li></ul> **_示例：_**&#x200B;由收入超过100万的产品Y帐户定义的帐户段 |
| 购买组角色模板 | 此组件提供以下问题的答案： <ul><li>您正在定位哪些角色？</li><li>哪组规则可用于确定分配给购买组角色的用户？</li></ul>  **_示例：_**&#x200B;将具有CMO标题的人员分配给决策者角色 |

## 购买组工作流程

1. 创建购买组。

   选项：
   * 使用[感兴趣的解决方案](./solution-interests.md)和[角色模板](./buying-groups-role-templates.md)
   * 使用第三方导入
   * 从AI/ML生成

1. 识别失踪人员。

   使用过滤器分析购买群体。

   **_示例：_**&#x200B;缺少决策者角色，完整性分数小于50

1. 完成购买组定义。

## 购买团体详细信息

要访问购买群组的详细信息，请在&#x200B;_[!UICONTROL 浏览]_&#x200B;选项卡中单击购买群组名称。

![购买团体详细信息](assets/buying-group-details.png){width="600" zoomable="yes"}

### 购买群组完整性分数

完整性得分用于确定购买组是否完整，这意味着它拥有分配给角色的正确成员，并准备好在帐户历程中使用。 此得分是一个百分比，它基于购买组内的角色数量以及至少分配了一个潜在客户的角色数量。

例如，如果购买组中有四个角色，且四个角色中的三个角色至少分配给一个潜在客户，则购买组的完成率为75%。

每次创建或更新购买群组时，都会重新计算购买群组完整性分数。

### 购买群组参与度得分

购买团体参与度得分是一个数字，用于根据购买团体成员执行的活动确定其参与度。 购买组成员在过去30天内执行的任何入站活动均用于计算得分。

每个活动的每日频率上限为20。 如果购买组的成员每天执行相同活动超过20次，则活动的数量上限为20次，而不是更高的数量。

显示的分数会四舍五入。 例如，得分75.89999显示为76。

#### 权重

用户可以为角色模板中的每个角色分配&#x200B;_权重_，为角色分配不同的权重以计算参与度得分。

![为角色模板中的每个角色设置权重](./assets/roles-templates-weighting.png){width="700" zoomable="yes"}

每个权重级别将转换为值，该值用于计算参与度分数：

* [!UICONTROL 普通] = 20
* [!UICONTROL 次要] = 40
* [!UICONTROL 正常] = 60
* [!UICONTROL 重要] = 80
* [!UICONTROL 重要] = 100

具有三个加权为&#x200B;_[!UICONTROL Vital]_、_[!UICONTROL Important]_&#x200B;和&#x200B;_[!UICONTROL Normal]_&#x200B;角色的角色模板将转换为以下加权百分比：

| 角色 | 权重 | 系统值 | 值计算 | 百分比 |
|-------------- |--------- |------------- |------------------ |---------- |
|               |          |              |                   |           |
| 决策者 | 重要 | 100 | 100/240 | 41.67% |
| 影响因素 | 重要 | 80 | 80/240 | 33.33% |
| 从业者 | 正常 | 60 | 60/240 | 25% |
|               | 总计 | 240 |                   |           |
