# 等待节点

当您想要在进入下一步之前暂停历程进程特定持续时间时，请使用&#x200B;_等待_&#x200B;节点。

您可以通过两种方式定义等待时间：

* 要前进到历程中的下一个节点的特定日期
* 相对持续时间（分钟数、小时数、天数、周数或月数）

## 添加等待节点

1. 导航到历程图。

1. 单击路径上的加号( **+** )图标，然后选择&#x200B;**[!UICONTROL 等待]**。

   ![添加历程节点 — 等待](./assets/add-node-wait.png){width="440"}

1. 在右侧的节点属性中，设置历程进入路径中的下一个节点之前等待的时间的&#x200B;**[!UICONTROL Type]**。

   * **[!UICONTROL 持续时间]** — 定义从进入到退出等待节点之间经过的特定天数、小时数或分钟数。
   * **[!UICONTROL 日期]** — 为退出指定特定的日期和时间。

   ![历程节点 — 等待](./assets/node-wait.png){width="500"}

## 高级等待设置

启用&#x200B;**[!UICONTROL 必须于]**&#x200B;结束选项以配置&#x200B;_高级等待步骤_，并确保您的邮件在最佳时刻到达人员和帐户成员。 此配置可让您精确控制人员或帐户何时退出等待步骤并进入历程中的下一个节点。 您可以计划在一周中的特定时间和特定日期执行的操作，而不是从进入到退出为止的固定小时数或天数。

通过&#x200B;_高级等待步骤_，您可以定义&#x200B;**_何时退出_**，而不只是他们应该等待多长时间。

![历程节点 — 高级等待步骤](./assets/node-wait-advanced.png){width="500"}

>[!AVAILABILITY]
>
>高级等待设置适用于在[!DNL Journey Optimizer B2B Edition]简化架构[上配置的](../simplified-architecture.md)环境。

### 等待类型

| 等待类型 | 描述 | 配置 |
| --------- | ----------- | ------------- |
| **一天中的特定时间** | 保持到特定时间（如上午9:00） | 设置时间（小时和分钟）。 在该时间的下一个事件中退出（对于所选时区）。 |
| **每周特定日期** | 保持到特定日期（如星期二） | 选择一周中的某一天。 如果未指定时间，则在下一个匹配日期的午夜（对于所选时区）退出。 |
| **天范围或组合** | 保持到某个范围（例如星期一至星期五）内的任何一天或指定日期的任何一天 | 选择您的目标日期。 如果未指定时间，则在下一个匹配日期的午夜（对于所选时区）退出。 |
| **时间+天组合** | 将两者合并以实现精确的时间安排（例如，星期二上午10:00） | 选择目标天数并设置目标时间。 在出现次日/时间时退出（对于所选时区）。 |

### 常见应用场景

以下方案说明了如何将典型方案应用于等待节点配置：

+++电子邮件在工作时间到达

**情景：**&#x200B;您向在工作时阅读电子邮件的B2B客户进行营销。 您希望所有电子邮件都在营业时间送达。

**解决方案：**&#x200B;配置您的等待步骤以在工作日上午9:00（星期一至星期五）释放潜在客户。 无论潜在客户何时进入等待节点，他们都会在工作时间收到您的电子邮件。

+++

+++动态受众的一致发送时间

**方案：**&#x200B;受众每天都会更改，因为新帐户或潜在客户符合条件。 您希望所有潜在客户都同时收到第一封电子邮件，而不管他们何时符合条件。

**解决方案：**&#x200B;将您的等待步骤设置为在特定时间（例如，上午10:00）结束。 所有潜在客户（无论是在午夜还是中午符合条件）均在上午10:00一起退出等待步骤。

+++

+++与SLA兼容的跟进任务

**方案：**&#x200B;您的销售团队有一个为期两个工作日的SLA来跟踪符合营销条件的客户潜在客户。 周末不算。

**解决方案：**&#x200B;将等待步骤配置为仅在工作日释放潜在客户。 周五获得资格的销售线索将在周一或周二进行后续追踪，而不是在周末进行。

+++

### 登入和退出示例

| 等待配置 | 客户/潜在客户输入 | 帐户/潜在客户退出 |
| ------------------ | ------------------- | ------------------ |
| 上午9:00，任何时间 | 星期一上午11:00 | 星期二上午9:00 |
| 上午9:00，任何时间 | 星期一上午7:00 | 星期一上午9:00 |
| 星期二，没有时间安排 | 星期五3:00下午 | 星期二上午12:00 |
| 上午10:00，星期一至星期五 | 星期六2:00下午 | 星期一上午10:00 |
| 上午10:00，星期一至星期五 | 星期三上午8:00 | 星期三上午10:00 |
