# 聊天界面

聊天面板在整个[!DNL Adobe Journey Optimizer B2B Prime]中嵌入。 在这里，您可使用简单语言与人工智能代理交互，构建项目和历程、创建和比较人员列表、调查潜在客户、配置得分等。

选择左侧导航中的&#x200B;_AI助手_&#x200B;图标以打开面板。 面板标题包含四个控件：

| 控件 | 描述 |
|---------|-------------|
| **新对话** | 开始新聊天（清除当前会话）。 |
| **对话历史记录** | 打开过去的对话，以便重新打开一个对话。 |
| **交换面板** | 将聊天面板移动到工作区的另一侧。 |
| **折叠** | 隐藏面板，为工作区提供更多空间。 |

面板底部是消息框，您可以在其中执行以下操作：

* 添加消息并按&#x200B;**Enter**&#x200B;发送（**Shift+Enter**&#x200B;插入新行）。
* 使用&#x200B;_附加_&#x200B;图标附加文件（支持的格式：`.txt`、`.md`、`.csv`、`.json`、`.xlsx`、`.docx`、`.pdf`）。 使用CSV和电子表格上传功能启动商机导入。

## 询问AI助手

有两种同样有效的方式完成工作 — 您永远不必使用斜杠菜单。

**自然语言** — 按照您对同事的陈述方式键入您的请求：

* _“创建新的试用注册欢迎历程。”_
* _“为什么john@acme.com没有进入此历程？”_
* _“比较我的‘第3季度网络研讨会与会者’和‘企业帐户’列表。”_

座席在幕后将您的措辞与适当的技能相匹配，并运行适当的工作流。

**正斜杠(/)菜单** — 键入`/`以打开助理可以执行的所有操作的可浏览菜单。 当您想要发现功能或直接跳转到已知的工作流时，这将很有用。

## 斜杠(/)菜单

在消息框开头或空格后使用`/`打开菜单。 在您继续键入时，列表将按名称、描述或ID进行筛选 — 例如，`/journey`将缩小为与历程相关的命令。

使用&#x200B;**/**&#x200B;在条目之间移动，使用&#x200B;**Enter**&#x200B;或&#x200B;**Tab**&#x200B;选择，使用&#x200B;**Esc**&#x200B;取消。 也可以直接单击一个条目。

菜单项被分组到标记部分中：

| 部分 | 包含 |
|---------|----------|
| **调查** | 诊断、只读查找（例如，潜在客户调查、意图查询）。 |
| **验证** | QA和审核工作流。 |
| **生成** | 创建工作流（项目、历程、列表、评分）。 |
| **导入** | CSV潜在客户导入。 |
| **其他** | 任何不属于上述类别的项目。 |

该菜单还列出了&#x200B;**连接器**（例如，_浏览Marketo_&#x200B;打开选取器对话框）和&#x200B;**导航快捷键**，可让您跳转到工作区屏幕。

### 选择菜单项

当您从菜单中选择技能或代理时，会向消息框中插入起始提示，供您编辑 — 它&#x200B;**未**&#x200B;自动发送。 例如，选择&#x200B;_计划促销活动_&#x200B;在&#x200B;_中丢弃“为[促销活动名称]计划新的Marketo计划”。 我将上传简报。”_ 填写方括号中的占位符并按&#x200B;**Enter**&#x200B;发送。

连接器会打开一个模式窗口，而不是插入文本。 导航快捷键会将您直接转到工作区中的该屏幕。

>[!NOTE]
>
>某些命令显示为灰色并标记为&#x200B;_即将推出_。 这些功能由功能标志限定，对于您的帐户尚未激活 — 选择一个不会产生任何效果。 可用的设置取决于为您启用的功能。

## 技能

技能是代理知道如何运行的打包工作流 — `/`菜单和自然语言请求背后的构建块。 每个技能都捆绑了分步说明和一个工作所需的特定工具（例如，“发布历程”、“比较两个人列表”、“构建评分模型”）。

关于技能，需要了解的关键事项：

* **技能属于产品范围。** 在AJO B2B Prime中，您可以看到AJO B2B技能（旅程、人员列表、评分、渠道、发送时间优化等）。 一些技能仅适用于Marketo，还有一些技能同时适用于这两种产品（潜在客户导入、产品知识）。 您只会看到与您所在位置相关的技能。
* **您不需要记住技能名称。** 描述您的目标，工程师会挑选匹配的技能。 `/`菜单是同一工作流的更快、可发现的快捷方式。
* **某些技能仅可读取；其他技能可更改内容。** 调查和报告技能（例如，商机调查、意图查询、发送时间报告）仅读取数据。 构建和配置技能（例如，历程创建、评分）以创建或更改数据。

## 跟进提示

助理回复后，它通常会根据您刚刚完成的任务显示一行可点击的跟进提示 — 例如，在创建历程后，它可能会提供&#x200B;_“发布此历程”_&#x200B;或&#x200B;_“添加等待步骤”_。 单击其中一个按钮可继续操作，而无需键入。 这些只是建议；您始终可以键入自己的下一封邮件。

## 提示

* **指定标识符。** 在调查或编辑时，请提供电子邮件地址、人员姓名、历程名称或列表名称，以便代理对正确的资产执行操作。
* **使用`/`浏览。** 如果不确定助理可以做什么，请打开`/`菜单并浏览类别。
* **编辑起始提示。** 选择技能可为您提供一个模板 — 在发送前替换`[bracketed]`占位符。
* **首先上载导入。** 对于潜在客户导入，请先附加CSV，然后说明您要对其执行的操作。
* **当您切换到不相关的任务时，启动新对话**，以便较早的上下文不会影响新请求。
