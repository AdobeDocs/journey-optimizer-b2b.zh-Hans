# Adobe Journey Optimizer B2B Prime文档

[!DNL Adobe Journey Optimizer B2B Prime]是一个人工智能优先的营销自动化解决方案，它使营销人员和业务开发代表能够规划项目、编排个性化人员历程并鉴别潜在客户。 使用对话式AI跨电子邮件和WhatsApp渠道构建营销活动、管理受众并提供内容。
